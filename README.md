# Kişisel Dijital Portfolyo

Bu proje, Kapadokya Üniversitesi Web Tasarımı dersi (BGY207) için hazırlanmış bir kuiz proje ödevidir. HTML5, CSS3 ve JavaScript kullanılarak oluşturulmuş bir "One-Page" kişisel portfolyo sitesidir.

## Proje Amacı
Öğrencilerin web teknolojilerini kullanarak kendilerini tanıtan bir site geliştirmeleri ve GitHub ile profesyonel geliştirme sürecini simüle etmeleri.

## Kullanılan Teknolojiler
- HTML5 (Semantik etiketler)
- CSS3 (Flexbox, Grid, Responsive Design)
- JavaScript (Form handling)

## Kurulum
1. Klasörü indirin.
2. `index.html` dosyasını tarayıcıda açın.
